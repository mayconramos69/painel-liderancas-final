# PAINEL LIDERANÇAS - PRONTO PARA GITHUB

## Como subir no GitHub

Envie para o repositório estes arquivos e pastas:

- app.py
- Procfile
- README.md
- requirements.txt
- runtime.txt
- static/
- templates/

IMPORTANTE:
No GitHub, o arquivo app.py precisa aparecer logo na primeira tela do repositório.
Não pode ficar dentro de outra pasta.

## Render

Build Command:
pip install -r requirements.txt

Start Command:
gunicorn app:app
